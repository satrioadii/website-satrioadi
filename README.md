# Website satrioadi.com

Personal website of mine (satrio adi) in order to promote my self. I want people
believe in me about my works in technology.

## Getting Started

These instructions will get you a copy of the project up and running on your
local machine for development. See deployment for notes on how to deploy the
project on a live system.

### Prerequisites

If you interested to clone this project, you may use this command.

```
git clone https://github.com/satrioadii/website-satrioadi.git
```

### Installing

A step by step series of examples that tell you how to get a development env
running

You can start the development server by enter this command after the clone
repositories succeed.

```
npm install
```

then run the server with this command

```
npm run dev
```

The development server is started on port 3000.

## Deployment

In order to deploy this project, run this code first/

```
npm run build
```

Then, you may use one of this reference as the development process in your
server.

- [Official Docs for Deployment from website](https://nextjs.org/docs/deployment) -
  Deployment using Vercel Platform
- [Setup Node JS App for Production](https://www.digitalocean.com/community/tutorials/how-to-set-up-a-node-js-application-for-production-on-ubuntu-16-04) -
  Deployment using Nginx on Ubuntu

Or search for your preferred options.
